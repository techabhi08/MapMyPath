Instructions

Click within the grid and drag your mouse to draw obstacles.

Drag the start node to set the start position.

Drag the end node to set the end position.

Choose an algorithm from the "Algorithms" drop-down menu.

You can also draw Mazes and Random walls by choosing "Maze Generation" and "Random Walls" from above drop-down menu.
